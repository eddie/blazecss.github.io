--- layout: page title: Badges ---

They're useful when you want to add additional info. Like a version number of a project or for status messages for individual items.

v{{ site.version }} 
<span class="c-badge">v{{ site.version }}</span>

Colours

Badges come as default, primary, secondary, success and error

To use, add the colour modifier class:

Default Primary Secondary Success Error 
<span class="c-badge">Default</span>
<span class="c-badge c-badge--primary">Primary</span>
<span class="c-badge c-badge--secondary">Secondary</span>
<span class="c-badge c-badge--success">Success</span>
<span class="c-badge c-badge--error">Error</span>

Rounded

Make your badges rounded with the .c-badge--rounded modifier.

Default Primary Secondary Success Error 
<span class="c-badge">Default</span>
<span class="c-badge c-badge--rounded c-badge--primary">Primary</span>
<span class="c-badge c-badge--rounded c-badge--secondary">Secondary</span>
<span class="c-badge c-badge--rounded c-badge--success">Success</span>
<span class="c-badge c-badge--rounded c-badge--error">Error</span>
